# Pop! The Question!
## Work in progress

A simple full-stack polling app viewable [here](pop-the-question.herokuapp.com).

Built with the Heroku-CRA-Node boilerplate to run the React front end and Express
back end on one server.

## Tools Used:

- Node.js
- Express
- MongoDB
- Passport.js
- React
- React-Router
- React-Bootstrap
